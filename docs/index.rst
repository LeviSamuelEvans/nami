nami documentation
==================

Nami is a lightweight research library for flow matching, stochastic interpolants,
and diffusion-style generative modelling.

Use these sections to get oriented quickly:

- :doc:`overview`: core design concepts and package layout
- :doc:`quickstart`: installation, local docs workflow, and minimal examples
- :doc:`components`: high-level map of distributions, solvers, paths, schedules, and losses
- :doc:`examples`: notebooks in `books/` for walkthroughs and different experiments
- :doc:`api/index`: complete generated API reference
